# GitClauder 会話レポート

このディレクトリには、全タスクの実行履歴を記録した統合レポートが保存されます。

## conversation-history.md

- 全タスクの実行履歴を時系列で記録
- 各タスクの指示内容、結果、SessionIDを含む
- 新しいタスクは常にこのファイルの履歴を読み込んで実行される
- SessionIDが違っても、全体の話題はこのレポートで把握できる

## SessionIDの役割

- **統合レポート**: 全タスクの履歴（SessionID関係なく全て記録）
- **SessionID**: Claude CLIの局所的な会話継続機能を使う場合に指定
  - 同じSessionIDを使うと、より詳細な文脈で会話が継続される
  - 基本は自動生成、どうしても特定の話の続きをする場合のみ手動指定
